Robust Design Optimization Package: RODOP+

About RODOP+

RODOP+ is a design optimization software, which is suitable for non-linear optimization problems, where the objective function and/or design constraints are evaluated by performing computationally expensive simulation processes.

  • Global constrained optimization
  • Machine learning based data-driven approach
  • Gradient enhancement
  • Mixed-integer optimization
  • Hyper-parameter optimization for expensive machine learning models
  • Constraints as black-box proccesses and/or user defined functions (UDFs)
